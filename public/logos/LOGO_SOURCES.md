## Best Sources for Logos

1. **SportsLogos.net** - https://www.sportslogos.net/
   - Has many international basketball logos
   - High quality, organized by league

2. **Wikimedia Commons** - https://commons.wikimedia.org/
   - Search: "[Team name] logo"
   - Usually has SVG versions
   - Free to use

3. **Official Team Websites**
   - Look for "Press", "Media", "Brand Assets", or "Downloads" sections
   - Usually highest quality

4. **EuroLeague Official Site** - https://www.euroleaguebasketball.net/
   - Team pages often have logos

## Logo Specifications

- **Format**: PNG with transparency (preferred) or SVG
- **Size**: 500x500 pixels minimum
- **File naming**: Use lowercase with hyphens (e.g., `real-madrid.png`)
- **Save to**: `public/logos/`

## Quick Download Tips

1. When you find a logo, right-click and "Save image as..."
2. Save directly to the `public/logos/` directory
3. Name the file exactly as shown in `lib/loadSchedules.ts` mapping
4. If the logo has a white background, try to find a transparent version
5. Resize to 500x500 if needed
